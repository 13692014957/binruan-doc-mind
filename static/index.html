👋 操作说明
1) 选择一个 .txt、.docx、.md 或 .pdf 文件（txt/md ≤ 1MB，docx/pdf ≤ 2MB）。
2) 点击「开始解析」，大模型会从文档中自动提取候选知识节点。
3) 在右侧编辑 / 勾选，最后点击「确认写入知识库」。
